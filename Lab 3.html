Statistical Question: What number of points lie outside the Gaussian distribution associated with fluctuations in temperature readings (mean = 12, scale = 0.4)? Note: this Gaussian was assumed/given, not from actual data.

Math: Need to make a choice on what "outside" the distribution means- we chose anything beyond 5 sigma (statistical significance threshold). The 5 sigma probability is a known value from Z-tables (~1/3.5e6), but you can use the cdf function to find it. Use the ppf then to find what actual values this probability corresponds to.
$ N(outside) = (\int_{-\infty}^{5\ sigma\ value} (Histogram) \ dx + \int_{5\ sigma\ value}^{\infty} (Histogram) \ dx)$
Using the frequency histogram, not the density histogram.

prob5 = 1 - stats.norm.cdf(5, scale = 1, loc = 0)
thresh_low = stats.norm.ppf(prob5, loc = 12, scale = 0.4)
thresh_high = stats.norm.ppf(1 - prob5, loc = 12, scale = 0.4)

d = np.append(stats.norm.rvs(loc = 12., scale = 0.4, size = 100000), [10., 10.3, 2.1, 0., 0., 15.6, 22.3, 12.7])

count_tot = len(d)

count_low = 0
for i in range(len(d)):
    if(d[i] <= thresh_low):
        count_low = count_low + 1
        
count_high = 0
for i in range(len(d)):
    if(d[i] >= thresh_high):
        count_high = count_high + 1
        
number_out = count_low + count_high
print("Lower threshold value for 5 sigma is " + str(thresh_low) + ". Upper threshold value for 5 sigma is " + str(thresh_high) + ".")
print("Number Outside: " + str(number_out) + ". Total Number: " + str(count_tot))

Statistical Question: What is the distance traveled associated with a 5 sigma detection?

Math: Distance is $(x^{2} +y^{2})^{1/2}$, so the distribution should follow the same. The square root of the squares of two Gaussians gives a Rayleigh, so we just need to find the 5 sigma value of a Rayleigh distribution (the scale of the sigma goes as the square root of the scale of the Gaussians, but this is still just 1).
Prob ~ 1/3.5e6 as before.
The wanted distance is the ppf of a Rayleigh using mean 0 and scale 1 with the probability from above.

Statistical Question: What is the significance associated with seeing 6,800 cosmic rays over 15 days in the shadow of the moon?

Math: Cosmic ray radiation from space generally follows a Poisson background distribution. The mean of the Poisson is just the normal mean (1) times the number of minutes we are scanning for (60 8 15 = 7200). Given this, we can integrate the Poisson density distribution to get the probability (integrate to left since the mean will be to the right of our value)- $Prob = \int_{-\infty}^{6800} pdf(x) \ dx$. Then we can use the ppf function to get the associated sigma value.
